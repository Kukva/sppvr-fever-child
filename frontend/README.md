# Мария Ивановна — Frontend

React TypeScript приложение для мультиагентной системы маршрутизации детей с лихорадкой.

## Особенности

- 🏥 **Медицинский интерфейс** - Специально разработанный для медицинских работников
- 🤖 **Мультиагентная система** - Отображение статуса работы ИИ-агентов в реальном времени
- 💬 **Интерактивный чат** - Общение с ИИ-ассистентом для получения рекомендаций
- 📋 **Умные формы** - Валидация медицинских данных с подсказками
- 📄 **PDF экспорт** - Генерация медицинских отчетов для печати
- 📱 **Адаптивный дизайн** - Работает на всех устройствах
- ⚡ **Real-time обновления** - WebSocket соединение для мгновенных обновлений

## Технологический стек

- **React 19** - Современный UI фреймворк
- **TypeScript** - Типобезопасность
- **Vite** - Быстрая сборка и разработка
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **React Router** - Маршрутизация
- **React Hook Form** - Управление формами
- **Socket.IO** - Real-time коммуникация
- **jsPDF** - Генерация PDF документов
- **Axios** - HTTP клиент

## Установка и запуск

### Требования

- Node.js 18+
- npm или yarn

### Локальная разработка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd fiber/frontend
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр сборки

```bash
npm run preview
```

## Docker

### Сборка Docker образа

```bash
docker build -t fever-assistant-frontend .
```

### Запуск контейнера

```bash
docker run -p 80:80 fever-assistant-frontend
```

## Переменные окружения

| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `VITE_API_BASE_URL` | URL API бэкенда | `http://localhost:8000` |
| `VITE_WS_URL` | WebSocket URL | `ws://localhost:8000` |
| `VITE_ENABLE_DEBUG` | Включить отладочные логи | `false` |
| `VITE_APP_TITLE` | Заголовок приложения | "Мария Ивановна" |

## Компоненты

### PatientForm
Форма ввода данных пациента с валидацией:
- Базовая информация (имя, возраст, вес, рост)
- Температура и симптомы
- Дополнительная информация

### ChatInterface
Интерактивный чат с ИИ-ассистентом:
- Отправка сообщений
- История переписки
- Индикаторы набора текста

### AgentStatus
Визуализация работы мультиагентной системы:
- Статус каждого агента
- Общий прогресс
- Управление процессом

### RecommendationsPanel
Панель с медицинскими рекомендациями:
- Сортировка по приоритету
- Категоризация рекомендаций
- Детальное описание

### SpecialistCards
Карточки специалистов:
- Фильтрация по специальности
- Контактная информация
- Рейтинги и отзывы

### PDFExport
Экспорт результатов в PDF:
- Полный отчет о консультации
- Медицинские рекомендации
- Информация о специалистах

## Хуки

### useWebSocket
Управление WebSocket соединением:
- Автоматическое переподключение
- Обработка ошибок
- Real-time обновления

### useApi
Унифицированная работа с API:
- Обработка ошибок
- Индикаторы загрузки
- Кэширование запросов
