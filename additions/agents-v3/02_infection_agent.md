# Валидация агента: ИНФЕКЦИОННЫЕ ПРИЧИНЫ ЛИХОРАДКИ
**Документ для клинической проверки СППВР «Лихорадящий ребёнок»**
*Версия 1.0 — Май 2026 | Статус: черновик*

---

## Что делает этот агент

Анализирует данные пациента и определяет вероятные инфекционные причины лихорадки. Работает после триажа. Охватывает: ОРВИ, грипп, пневмонию, отит, ИМП, ОКИ, менингококковую инфекцию, сепсис, эндокардит, туберкулёз.
