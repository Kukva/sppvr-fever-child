# Мастер-документ: Все тест-кейсы с ожидаемыми ответами
**Итоговый документ для клинической валидации СППВР «Лихорадящий ребёнок»**
*Версия 1.0 — Май 2026 | Статус: черновик*

---

## Инструкция для врача-валидатора

1. Прочитайте сценарий кейса
2. Запустите кейс через систему СППВР
3. Сравните ответ системы с ожидаемым
4. Заполните таблицу оценки
5. Укажите замечания в свободном поле

**Шкала оценки каждого ответа:**
- **PASS** — ответ совпадает по зоне маршрутизации и ключевым диагнозам
- **MINOR** — зона правильная, но детали неточные или неполные
- **FAIL** — неверная зона или пропущен ключевой диагноз
