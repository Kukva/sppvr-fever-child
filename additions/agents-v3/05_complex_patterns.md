# Валидация агента: СЛОЖНЫЕ ПАТТЕРНЫ — ЛИХОРАДКА + ВЕДУЩИЙ СИМПТОМ
**Документ для клинической проверки СППВР «Лихорадящий ребёнок»**
*Версия 1.0 — Май 2026 | Статус: черновик*

---

## Что делает этот агент

Разбирает нетипичные сложные паттерны, при которых стандартный триаж недостаточен. Вызывается оркестратором при наличии ведущего симптома помимо лихорадки. Охватывает: лихорадка + хромота, лихорадка + боли в животе, лихорадка на антибиотиках, лихорадка + эндокардит.
