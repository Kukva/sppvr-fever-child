# ЗАДАЧА АГЕНТА
Извлечь и структурировать данные о пациенте из диалога с врачом. Если данных недостаточно — сформировать список обязательных вопросов. Источники: КР «ОРВИ у детей» (КР25), КР «ЛБОИ» (Союз педиатров России), КР «Сепсис у детей» 2025.

---

# ОБЯЗАТЕЛЬНЫЕ ПОЛЯ (без них продолжение невозможно)

## Базовые
- **Возраст** (в месяцах до 2 лет, затем в годах) — критически важен для пороговых значений
- **Пол**
- **Масса тела** (для дозировок)

## Лихорадка
- **Максимальная температура** (°С, способ измерения: ректально/аксиллярно/тимпанально)
- **Длительность лихорадки** (дней/часов)
- **Характер**: постоянная / волнообразная / двухволновая (важно для ГМИ)
- **Ответ на жаропонижающие**: снижается / не снижается / снижается затем повторный подъём
- **Жаропонижающее**: что давали, доза, время последнего приёма

## Сопутствующие симптомы (активно уточнять)
- Сыпь: есть/нет → если есть: когда появилась, где, характер (пятна/петехии/везикулы), динамика
- Катаральные явления: насморк, кашель, першение в горле
- Рвота: количество эпизодов, содержимое
- Диарея
- Боль в животе
- Головная боль
- Боль в ушах
- Боль при мочеиспускании, изменение цвета мочи
- Боли в конечностях, суставах
- Нарушение сознания/поведения: стал вялым, не реагирует, не узнаёт, раздражителен
- Нарушение дыхания: одышка, учащённое дыхание, шумное дыхание
- Отёки

## Анамнез
- **Прививочный статус**: вакцинирован по календарю / нет / частично (особенно: менингококк, пневмококк, Hib)
- **Контакты**: больные в окружении, посещение садика/школы, контакт с инфекционными больными
- **Эпидемиологический анамнез**: выезд за рубеж (куда, когда), контакт с животными
- **Хронические заболевания**: онкология, иммунодефицит, ВПС, БЛД, диабет, другие
- **Принимаемые препараты**: иммуносупрессоры, химиотерапия, антибиотики до обращения (если да — какие, сколько дней)
- **Предыдущие эпизоды**: аналогичные лихорадки в прошлом (важно для периодических синдромов)

## Объективный статус на момент осмотра
- Общее состояние: удовлетворительное / средней тяжести / тяжёлое
- Активность: активен / вялый / не реагирует
- Пьёт: да / отказывается / только с трудом
- Цвет кожи: обычный / бледный / мраморный / серый / цианотичный
- Сыпь при осмотре: описание
- ЧДД (если измерена)
- ЧСС (если измерена)
- SpO₂ (если измерена)
